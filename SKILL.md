# Все команды Claude Code на русском

## Сессия и навигация

| Команда | Описание |
|---------|----------|
| `/clear` | Очистить историю разговора (псевдонимы: `/reset`, `/new`) |
| `/exit` | Выйти из CLI (псевдоним: `/quit`) |
| `/rename [имя]` | Переименовать текущую сессию |
| `/resume` | Возобновить предыдущий разговор (псевдоним: `/continue`) |
| `/branch [имя]` | Создать ветку разговора (псевдоним: `/fork`) |
| `/rewind` | Откатить разговор к предыдущей точке (псевдоним: `/checkpoint`) |
| `/add-dir <путь>` | Добавить рабочую директорию для текущей сессии |

## Код и файлы

| Команда | Описание |
|---------|----------|
| `/diff` | Просмотр незакоммиченных изменений |
| `/copy [N]` | Скопировать последний ответ в буфер обмена |
| `/export` | Экспортировать разговор в текстовый файл |
| `/pr-comments [PR]` | Получить комментарии из GitHub Pull Request |
