# Claude Skill: help-ru

Скилл для [Claude Code](https://claude.ai/code), который выводит полный справочник всех команд Claude Code на русском языке.

## Что делает этот скилл

При вызове `/help-ru` Claude выводит в чат таблицы со всеми доступными командами Claude Code на русском языке, сгруппированными по категориям:

- Сессия и навигация
- Код и файлы
- Настройки
- Инструменты и интеграции
- Агенты и автоматизация
- Контекст и оптимизация
- Информация и статистика
- Аккаунт и план
- Разное
- Устройства и платформы
- Быстрые режимы ввода (`/`, `!`, `@`)

## Установка

1. Скопируйте файл `SKILL.md` в директорию `~/.claude/skills/help-ru/SKILL.md`
2. Перезапустите Claude Code
3. Вызывайте скилл командой `/help-ru`

## Использование

```
/help-ru
```

Claude выведет полный справочник команд прямо в чат.

## Совместимость

Протестировано с Claude Code CLI и десктоп-приложением.
